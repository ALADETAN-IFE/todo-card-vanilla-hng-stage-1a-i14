# Frontend Wizards Todo Card - Vanilla HTML/CSS/JS

This folder now contains a plain HTML, CSS, and JavaScript version of the Stage 0 Todo Card.

## Files

- `index.html` - Static card markup and accessibility hooks
- `style.css` - Layout, theme, and responsive styling
- `script.js` - Checkbox, time remaining, and button behavior
- `task.md` - Stage 1a task requirements
- `readme.md` - Setup and project notes

## What Works

- All Stage 0 `data-testid` hooks are preserved
- Checkbox toggles completion state
- Time remaining updates automatically
- Edit and Delete buttons are wired for basic interaction
- Responsive layout works from mobile to desktop

## Run Locally

Open `index.html` in a browser, or serve the folder with any static server.

If you want to use a local server:

```bash
npx serve .
```

## Notes

- This is the conversion layer only.
- Stage 1a-specific edit mode, status controls, expand/collapse, and overdue enhancements will be added after you confirm this version is okay.
